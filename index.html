Publiceer via Pages

GitHub Pages zet je bestanden uit de repository om naar een publieke website.

Publicatie

Van repository naar live URL

  1. Zet de repository op public.
  2. Ga naar Settings → Pages.
  3. Kies Deploy from branch.
  4. Selecteer main en root.
  5. Open de GitHub Pages URL.
